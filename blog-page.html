Important FAQ

What is the DOM?

The DOM is an interface that represents the structure of an HTML or XML document as a tree of objects.

How can you access an element in the DOM using JavaScript?

access an element using document.getElementById(), document.querySelector(), or document.getElementsByClassName().

What is the difference between innerHTML and textContent?

innerHTML returns or sets the HTML content of an element, while textContent only handles plain text.

How do you add a new element to the DOM?

create an element with document.createElement() and append it using appendChild() or insertBefore().
